Colocando links no texto sem precisar de abrir outra pagina

Para abrir o link em uma nova aba basta adicionar um TARGET com um branco (_blank) e um Rel ou relacionado(rel)

observações

Normalmente colocamos o target blank quando queremos mandar a pessoa para um site que não é nosso , assim mantemos a aba do nosso site caso o cliente queira voltar

já sem o target normalmente usamos para quando queremos navegar no nosso proprio site
